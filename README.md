## Neovim

im using nvchad for nvim configuration, you can check on [nvchad.com](https://nvchad.com/) for installation.
after installing, you can copy and paste the configuration to ``.config/nvim``

other plugin are used

- wakatime
- markview
- triforce
- todo.nvim
- trouble
